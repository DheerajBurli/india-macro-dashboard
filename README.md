# India Macro Dashboard

High-frequency GDP component tracker — 40 indicators across 5 buckets (C / I / G / X-M / Cross-Cutting) with next-quarter ML predictions.

## Stack
- **React 18** + **Vite** (frontend)
- **Chart.js 4** via `react-chartjs-2` (charts)
- **Vercel** (deployment)
- **ML Pipeline** — see `src/ml/README.md` (currently dummy/placeholder)

## Quick Start

```bash
# 1. Install
npm install

# 2. Dev server
npm run dev

# 3. Build
npm run build
```

## Deploy to Vercel

### Option A — Vercel CLI
```bash
npm i -g vercel
vercel login
vercel --prod
```

### Option B — GitHub Integration (recommended)
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import repo
3. Framework: **Vite** (auto-detected)
4. Build command: `npm run build`
5. Output dir: `dist`
6. Click Deploy ✓

---

## Wiring in Real Data

### 1. Historical series
Edit `src/data/indicators.js`. For each indicator, replace:
```js
actuals: nulls(24)
```
with your actual time-series array (same length as `labels`).

### 2. ML Predictions
See `src/ml/README.md` for two options:
- **Option A**: Drop `public/predictions.json`, fetch in `App.jsx`
- **Option B**: Vercel Edge Function at `api/predict.js`
